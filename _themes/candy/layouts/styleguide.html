Adding the colour classes white, chocolate, red, redFaded, yellow, yellowFaded, teal, or tealDark will change the icon's colour.

Buttons

Every button has the .btn class attached to it.

To change the colour from the default red, these colour classes can be used: redFaded, yellow, yellowFaded, teal, tealDark, and chocolate.

Examples

<button class="btn">Default button</button>
<button class="btn yellow">Yellow button <i class="icon-pointer-right" aria-hidden="true"></i></button>
<button class="btn chocolate">Mmmm, chocolate <i class="icon-heart" aria-hidden="true"></i></button>
